## 项目组成

- **浏览器扩展**（`/extension`）：Manifest V3，兼容 Chrome / Edge，Firefox 降级支持。
- **产品落地页**（`/frontend`）：React 单页应用，介绍产品并提供扩展。
- **中文文档**（`/docs`）：设计与技能文档归档。
