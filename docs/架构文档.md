## 二、扩展抓取原理（核心）

### 组合式抓取策略

1. **资源树抓取（等同 DevTools Sources 面板）**
   - `chrome.debugger.attach(tabId, "1.3")` 连接调试协议。
   - `Page.enable` → `Page.getResourceTree` 获取帧树与各帧资源列表。
   - 递归遍历 `frameTree`（含 `childFrames`），收集每个帧的文档 URL 与 `resources`。
   - 对每个资源调用 `Page.getResourceContent({frameId, url})` 获取内容（文本或 base64）。
   - 这一步确保得到的资源集合与「源代码 / 来源」面板中展示的树**一致**。

2. **DOM 解析补全**
   - `chrome.scripting.executeScript` 注入函数到页面上下文。
   - 收集 `img[src] / img/source[srcset] / script[src] / link[href] / video/audio/source/track / 内联 style 中的 url()` 引用的绝对 URL。
   - 对资源树中缺失的项使用 `fetch()` 补抓（受同源/CORS 限制的跨域资源会被安全跳过）。

3. **渲染 HTML 留存（可选）**
   - 保存 `document.documentElement.outerHTML` 作为渲染后的最终 HTML。

### 目录结构还原

- `urlToPath(url)`：以 `hostname + decodeURIComponent(pathname)` 作为 zip 内相对路径。
- 以 `/` 结尾或为空的路径补 `index.html`。
- 带查询串的资源在扩展名前追加 `_<短哈希>`，避免同名覆盖。
- 过滤 Windows 非法文件名字符。

### 打包与下载

- 使用 **JSZip** 增量 `zip.file(path, content, {base64})`。
- `zip.generateAsync({type:"blob", compression:"DEFLATE"})` 生成压缩包。
- `URL.createObjectURL(blob)` + `chrome.downloads.download()` 触发下载。
- 文件名：`sitezip_<host>_<时间戳>.zip`。

### 浏览器兼容

- **Chrome / Edge**：完整支持（含 `chrome.debugger`）。
- **Firefox**：`chrome.debugger` 不可用，自动降级为「DOM 解析 + fetch」抓取；manifest 已声明 `browser_specific_settings.gecko`。

## 三、扩展目录结构

```
/extension
├── manifest.json      # MV3 清单，权限: debugger/downloads/activeTab/scripting/tabs/storage
├── background.js      # 轻量 service worker
├── popup.html         # 弹窗 UI
├── popup.css          # 弹窗样式（Swiss/高对比暗色）
├── popup.js           # 核心抓取 + 打包逻辑
├── lib/jszip.min.js   # 本地打包库（MV3 禁止远程脚本）
└── icons/             # 16/48/128 图标
```

## 四、落地页架构

```
/frontend/src
├── App.js                     # 页面装配 + 全局背景
├── components/site
│   ├── Navbar.jsx             # 顶栏
│   ├── Hero.jsx               # 首屏
│   ├── Features.jsx           # Bento 功能网格
│   ├── HowItWorks.jsx         # 使用步骤
│   ├── Install.jsx            # 扩展安装 + 下载
│   ├── Disclaimer.jsx         # 免责声明
│   └── Footer.jsx             # 页脚
└── components/ui/*            # shadcn/ui 组件
```

- **技术栈**：React 19 + React Router 7 + Tailwind CSS 3 + shadcn/ui + framer-motion + lucide-react。
